membuat sebuah folder kosong dengan namamu sendiri ($git init Ardika)
membuat sebuah file di dalam folder tersebut dengan nama README.md, isi file tersebut dengan kalimat "Halo perkenalkan aku halaman utama" ($cd Ardika,$touch README.md,$vim README.md)       
inisialisasi folder tersebut dengan Git, kemudian simpan perubahan menggunakan commit dengan pesan "First commit" ($git init README.md,$git commit -m "First commit")
Ganti teks sebelumnya dengan `"Hello world" ($vim README.md,$"Hello world")
Tampilkan isi teks tersebut pada command line menggunakan command cat ($cat README.md)
Ternyata kamu tidak ingin perubahan tersebut, dan ingin kembali ke perubahan seperti commit yang terakhir. Lakukan teknik git backtracking untuk mengembalikan ke perubahan commit yang terakhir. ($git checkout HEAD README.md)
buat branch baru dengan nama cv, hal ini berguna agar histori kita tidak tercampur ($git branch cv)
pindah branch ke dalam cv, kemudian buat file dengan nama cv.txt dan isi file tersebut dengan kalimat:"Ini adalah file CV" ($git checkout cv,$touch cv.txt,$vim cv.txt,"Ini adalah file CV")
kemudian simpan perubahan menggunakan commit dengan pesan"Initial CV" ($git add .,$git commit -m "initial cv")
tambahkan 3 perusahaan yang akan kamu lamar, dan setiap menuliskan 1 nama perusahaan kamu harus melakukan dokumentasi dan menyimpan perubahan menggunakan commit (&touch youtube.txt && youtube.txt, $git add .,$git commit -m "menambahkan perusahaan pertama",&touch google.txt && google.txt, $git add .,$git commit -m "menambahkan perusahaan kedua"&touch facebook.txt && facebook.txt, $git add .,$git commit -m "menambahkan perusahaan ketiga")
kembali ke branch master ($ git checkout master) 
ubah file README.md jangan lupa untuk menyimpan perubahan menggunakan commit dengan pesan "update master pertama" ($ vim README.md, "Halo perkenalkan aku halaman utama Ini adalah update pertama pada branch master", $ git add .,$git commit -m "update master pertama")
gabungkan branch cv ke dalam branch master menggunakan perintah git merge ($ git merge cv)
unggah Git Repository project tersebut tersebut ke dalam GitHub ($ git remote add odessy https://github.com/ArdikaPA/piogg.git, $git remote -v, $git push odessy )
